<objective>
**What:** Create `src/experiments/trace_error_propagation.py` -- a single experiment script that loads the FP16 baseline checkpoint, runs one FP16 reference forward pass through the full model to capture clean P-point activations via ErrorPropagationTracker, then individually quantizes each of 21 source weight matrices (7 per layer: q_proj, k_proj, v_proj, o_proj, gate_proj, up_proj, down_proj across layers 0, 5, 11) using FP4 E2M1 round-to-nearest per-channel quantization, re-runs the forward pass with a fresh tracker for each source to capture perturbed P-points, computes per-P-point relative error ||delta||/||y|| at the source's own layer, produces error waterfall sequences, computes RMSNorm attenuation ratios and parallel/orthogonal error decomposition across all 12 layers, prints formatted waterfall tables, and exports structured JSON to `results/error_propagation_trace.json`.

**Purpose:** Determine where quantization error amplifies and where it attenuates through the Transformer layer pipeline. Answer the key numerical analysis question: do large-kappa matrices produce errors that survive RMSNorm attenuation, or does RMSNorm effectively normalize away all error regardless of source?

**Output:** `src/experiments/trace_error_propagation.py` (~500 lines), `results/error_propagation_trace.json` (generated at runtime).

```python
def _classify_matrix(module_path: str) -> tuple[int, str]:
    """Parse module path into (layer_index, matrix_type)."""
    parts = module_path.split(".")
    layer_idx = -1
    if "layers" in parts:
        idx = parts.index("layers") + 1
        if idx < len(parts):
            layer_idx = int(parts[idx])
    last_seg = parts[-1] if parts else "unknown"
    if last_seg in ("q_proj", "k_proj", "v_proj", "o_proj"):
        matrix_type = "attention"
    elif last_seg in ("gate_proj", "up_proj", "down_proj"):
        matrix_type = "ffn"
    elif last_seg in ("embed_tokens", "lm_head"):
        matrix_type = "global"
    else:
        matrix_type = last_seg
    return layer_idx, matrix_type
```

**1. `_compute_rmsnorm_metrics(all_rmsnorm_data, ref_p_points) -> dict`:**

For each per-source entry in `all_rmsnorm_data`:
- Extract `source_matrix`, `source_layer`, `q_p_points` dict
- For each layer_idx in 0..11:
  - **input_norm** (P0->P1 transition, per D-06):
    - `ref_P0` = ref_p_points[f"{layer_idx}_P0"]; `q_P0` = q_p_points[f"{layer_idx}_P0"]
    - `ref_P1` = ref_p_points[f"{layer_idx}_P1"]; `q_P1` = q_p_points[f"{layer_idx}_P1"]
    - `d_pre = q_P0 - ref_P0` (per D-07: norm input error)
    - `d_post = q_P1 - ref_P1` (per D-07: norm output error)
    - `d_pre_norm = d_pre.norm().item()`, `d_post_norm = d_post.norm().item()`
    - `ratio = d_post_norm / max(d_pre_norm, 1e-12)` (per D-07: attenuation ratio)
    - If `d_pre_norm < 1e-8` (no measurable error before norm, e.g., source layer's P0), mark ratio as NaN
  - **post_attn_norm** (P3->P4 transition, per D-06):
    - `ref_P3` = ref_p_points[f"{layer_idx}_P3"]; `q_P3` = q_p_points[f"{layer_idx}_P3"]
    - `ref_P4` = ref_p_points[f"{layer_idx}_P4"]; `q_P4` = q_p_points[f"{layer_idx}_P4"]
    - `d_pre = q_P3 - ref_P3`, `d_post = q_P4 - ref_P4`
    - Same ratio computation as input_norm
  - **input_norm decomposition** (TRACE-03, per D-08):
    - `y` = ref_P1 tensor (clean norm output), flattened to 1D: `y_flat = y.reshape(-1)`
    - `d` = q_P1 - ref_P1 tensor (error at norm output), flattened: `d_flat = d.reshape(-1)`
