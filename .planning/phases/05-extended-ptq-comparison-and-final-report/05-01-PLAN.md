    **Per-checkpoint activation capture** — captured ONCE per checkpoint at FP16, before any quantization:

    Function `capture_fp16_activations(model, data_dir, args, device)`:

    **Manual per-matrix error computation** — avoids Pitfall 1 (quantizer mismatch with GPTQ/Lloyd-Max):

    Function `compute_per_matrix_errors(original_weights, quantized_model, saved_activations, device)`:
    - `errors = {}`
    - For each `name, W_fp` in `original_weights.items()`:
      - `module_path = name.replace('.weight', '')`
      - If `module_path not in saved_activations`: continue
      - `x = saved_activations[module_path].to(device)`
      - Get `W_q` from `quantized_model` by resolving the parameter path (use `dict(quantized_model.named_parameters())[name]`)
      - `y_fp = x @ W_fp.T`
      - `y_q = x @ W_q.to(device).T`
      - `denom = y_fp.norm().clamp(min=1e-12)`
      - `errors[module_path] = ((y_q - y_fp).norm() / denom).item()`
    - Return `errors`
