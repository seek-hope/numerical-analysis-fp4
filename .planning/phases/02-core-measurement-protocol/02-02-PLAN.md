<objective>
Create `src/experiments/measure_qerror.py` -- a standalone experiment script that loads the Micro-Gemma-FP model from a checkpoint, attaches ErrorPropagationTracker, runs a single forward pass on validation data, computes per-matrix output-space error ||dy||/||y|| using FP4 round-to-nearest quantization, computes per-matrix weight-space relative error ||W_q - W_fp|| / ||W_fp|| (dw_norm), computes per-matrix kappa(W) via exact SVD, validates the pipeline with a null measurement, and outputs a structured results table.

Purpose: Provides the end-to-end measurement pipeline that executes Phase 2's goal -- a validated measurement of per-matrix output-space relative error AND weight-space relative error. This script is the runnable entry point for Phase 2 verification and produces the data that Phase 3 consumes.

Output: `src/experiments/measure_qerror.py` with CLI interface, producing printed results table and optional JSON output.
</objective>

Import section: from src.analysis.error_propagation import ErrorPropagationTracker, from src.analysis.condition import compute_all_condition_numbers, from src.quantization.fp_quantizer import FPQuantizer, from src.model.config import MicroGemmaFPConfig, from src.model.transformer import MicroGemmaFPForCausalLM, from src.experiments.training_utils import get_dataloader, load_checkpoint, plus torch, argparse, json, os.

    2. Model loading:
       - config = MicroGemmaFPConfig()
       - model = MicroGemmaFPForCausalLM(config)
       - device = torch.device(args.device if torch.cuda.is_available() else 'cpu')

       - model.to(device).eval()
       - Print: "Model loaded from {args.checkpoint} on {device}"
       - Print parameter count via model.count_parameters()

    3. Tracker setup and single forward pass:
       - tracker = ErrorPropagationTracker()
       - tracker.attach(model)
       - torch.manual_seed(42)
       - dataloader = get_dataloader(batch_size=1, max_seq_len=512, split='val', data_dir=args.data_dir)
       - batch = next(iter(dataloader))
       - input_ids = batch['input_ids'].to(device)
       - attention_mask = batch.get('attention_mask'); if present, move to device
       - with torch.no_grad(): model(input_ids, attention_mask=attention_mask)
       - tracker.detach()
       - tracker.compute_p3_p6()
       - Print activation count: "Captured {len(tracker.activations)} Linear layer activations"
       - Print P-point count: "Captured {len(tracker.p_points)} measurement points"

    4. Kappa computation:
       - kappas_raw = compute_all_condition_numbers(model)
       - Strip '.weight' suffix: kappas = {k.replace('.weight', ''): v for k, v in kappas_raw.items()}
       - Print: "Computed kappa for {len(kappas)} matrices"

    5. Error computation (both output-space and weight-space):
       - quantizer = FPQuantizer(fmt='fp4_e2m1', per_channel=True)
       - errors = tracker.compute_output_error(model, quantizer)
       - Print: "Computed ||dy||/||y|| for {len(errors)} matrices"
       - Compute dw_norm (weight-space relative error, per MEAS-04):
         Iterate over get_quantizable_weights() results. For each (name, W_fp):
         - W_q = quantizer.quantize(W_fp.data)
         - dw_norm_val = (W_q - W_fp.data).norm().item() / W_fp.data.norm().item()
         - Store in dw_norms dict with key = name.replace('.weight', '') (matches errors/kappas key convention)
       - Print: "Computed ||dW||/||W|| for {len(dw_norms)} matrices"

    6. Null measurement:
       - Print "Running null measurement validation..."
       - try: max_null_err = tracker.validate_null_measurement(model); print(f"Null measurement: max error = {max_null_err:.2e} -- PASS")
       - except ValueError as e: print(f"Null measurement: FAILED -- {e}"); sys.exit(1)

    Do NOT print the results table or save JSON in this task -- those are Task 2.
